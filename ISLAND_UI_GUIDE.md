# 🏝️ 灵动岛界面使用指南

## 界面概览

灵动岛是游戏的主控制中心，集成了音乐选择和游戏设置功能。

## 功能说明

### 📍 位置
- 屏幕顶部中央
- 显示当前播放的歌曲名

### 🎯 操作方式

#### 1. 展开灵动岛
- **点击**灵动岛胶囊
- 界面向下展开，显示完整功能

#### 2. 标签页切换
展开后顶部有两个标签：

**🎵 音乐**
- 横向滚动的歌曲列表
- 点击歌曲封面切换曲目
- 当前播放的歌曲有白色高亮边框

**⚙️ 设置**
- 画质设置（4档可选）
- 更多设置选项（未来扩展）

#### 3. 收起灵动岛
- 点击灵动岛外的任意位置
- 自动收起，恢复紧凑模式

## 标签页详解

### 🎵 音乐页面

**布局：**
```
┌─────────────────────────────┐
│  🎵 音乐  |  ⚙️ 设置        │ ← 标签切换
├─────────────────────────────┤
│  ┌───┐  ┌───┐  ┌───┐       │
│  │🎵 │  │🎵 │  │🎵 │  →    │ ← 横向滚动
│  └───┘  └───┘  └───┘       │
│  歌曲1   歌曲2   歌曲3       │
├─────────────────────────────┤
│ 点击曲目切换 · 点击空白处关闭 │
└─────────────────────────────┘
```

**功能：**
- 查看所有可用歌曲
- 点击切换歌曲
- 当前歌曲有视觉反馈

**布局：**
```
┌─────────────────────────────┐
```

- 未来可扩展更多设置

## 设计特点

### 🎨 视觉设计
- **毛玻璃效果**：半透明背景 + 模糊
- **渐变高亮**：选中项使用紫色渐变
- **流畅动画**：展开/收起/切换都有过渡

### 📱 响应式设计
- **桌面端**：宽度最大 600px
- **移动端**：宽度 95%，自适应屏幕
- **横屏优化**：自动调整布局

### ♿ 交互优化
- **点击反馈**：悬停/点击有视觉变化
- **防误触**：点击内容不会关闭界面
- **流畅滚动**：音乐列表支持触摸滚动

## 使用技巧

### 💡 快速切歌
1. 点击灵动岛
2. 直接点击想听的歌（默认在音乐页面）
3. 自动开始播放

### 💡 调整画质
1. 点击灵动岛
2. 切换到"设置"标签
3. 选择画质模式
4. 点击外部关闭

### 💡 查看当前状态
- 灵动岛收起时：显示歌曲名
- 设置页面：显示当前画质
