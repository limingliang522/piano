# 🎨 用户界面结构

## 📱 灵动岛（Dynamic Island）

灵动岛是游戏的主要交互界面，采用类似 iPhone 14 Pro 的灵动岛设计。

### 🎯 三个标签页

#### 1️⃣ 音乐标签（🎧）
**功能：** 选择和管理游戏音乐

**内容：**
- 🔍 搜索框：快速查找曲目
- 📜 横向滚动列表：浏览所有可用曲目
- 🎲 随机选择按钮：随机选择一首曲目
- ✕ 关闭按钮：收起灵动岛

**操作：**
- 点击曲目封面选择音乐
- 左右滑动浏览更多曲目
- 点击🎲随机选择
- 当前播放的曲目会高亮显示

---

#### 2️⃣ 用户标签（👤）
**功能：** 查看和管理用户信息

**内容：**
- 👤 用户信息卡片
  - 显示当前登录的用户名（渐变色）
  - 未登录时显示"未登录"
- 🚪 退出登录按钮
  - 仅在已登录时显示
  - 点击后返回登录界面

**操作：**
- 查看当前登录状态
- 点击"🚪 退出登录"切换账号

---

#### 3️⃣ 设置标签（🔧）
**功能：** 调整游戏设置

**内容：**
- 🔊 音量控制
  - 滑块：0-100%
  - 实时显示当前音量百分比
- ✕ 关闭按钮：收起灵动岛

**操作：**
- 拖动滑块调整音量
- 音量变化立即生效

---

## 🔐 认证界面

### 登录/注册模式
当用户未登录时，灵动岛会自动展开并显示认证界面。

#### 🎹 标题
- 渐变色标题："🎹 钢琴跑酷"
- 醒目的品牌展示

#### 📑 标签切换
- **登录标签**：已有账号的用户
- **注册标签**：新用户创建账号

#### 📝 登录表单
- 👤 用户名输入框
- 🔒 密码输入框
- 登录按钮
- 实时错误/成功提示

#### 📝 注册表单
- 👤 用户名输入框（至少3个字符）
- 🔒 密码输入框（至少6个字符）
- 🔒 确认密码输入框
- 注册按钮
- 实时错误/成功提示

---

## 🎮 界面状态

### 未登录状态
```
灵动岛（收起）
├─ 标题：点击登录或注册
└─ 点击展开 → 显示认证界面
```

### 已登录状态
```
灵动岛（收起）
├─ 标题：欢迎，[用户名]
└─ 点击展开 → 显示三个标签页
    ├─ 🎧 音乐
    ├─ 👤 用户
    └─ 🔧 设置
```

### 游戏中状态
```
灵动岛（收起）
├─ 标题：[当前曲目名称]
└─ 点击展开 → 暂停游戏，显示菜单
```

## 🎨 视觉设计

### 配色方案
- **主色调**：深色背景（rgba(0, 0, 0, 0.85)）
- **强调色**：紫蓝渐变（#667eea → #764ba2）
- **文字色**：白色（#fff）
- **错误色**：红色（#ef4444）
- **成功色**：绿色（#4ade80）

### 效果
- 🌫️ 毛玻璃背景（backdrop-filter: blur(20px)）
- ✨ 平滑过渡动画（transition: all 0.3s）
- 🎭 悬停效果（hover states）
- 📱 响应式设计（适配各种屏幕）

### 图标
- 🎧 音乐
- 👤 用户
- 🔧 设置
- 🔍 搜索
- 🎲 随机
- 🚪 退出
- 🔊 音量
- 🔒 密码
- ✕ 关闭

## 📱 交互流程

### 首次使用
1. 打开游戏
2. 自动显示认证界面
3. 选择"注册"标签
4. 填写用户名和密码
5. 点击"注册"
6. 注册成功，显示欢迎信息
7. 自动收起，进入游戏

### 再次使用
1. 打开游戏
2. 自动恢复登录状态
3. 显示欢迎信息
4. 点击灵动岛选择音乐
5. 开始游戏

### 切换账号
1. 点击灵动岛
2. 切换到"👤"用户标签
3. 点击"🚪 退出登录"
4. 自动显示登录界面
5. 输入新账号信息
6. 登录成功

---

## 🎯 设计原则

1. **简洁优先**：每个标签页只显示核心功能
2. **清晰分区**：功能明确分类，不混杂
3. **视觉反馈**：所有操作都有即时反馈
4. **流畅动画**：平滑的过渡效果
5. **响应式**：适配各种设备和屏幕
