# 🚀 快速参考

## 📱 灵动岛标签页

| 图标 | 名称 | 功能 |
|------|------|------|
| 🎧 | 音乐 | 选择曲目、搜索、随机选择 |
| 👤 | 用户 | 查看用户名、退出登录 |
| 🔧 | 设置 | 调整音量 |

## 🔐 认证系统

### 注册要求
- ✅ 用户名：至少 3 个字符
- ✅ 密码：至少 6 个字符
- ✅ 确认密码：必须匹配

### 登录方式
- 🖱️ 点击"登录"按钮
- ⌨️ 按回车键

## 🎮 游戏操作

### 💻 电脑
| 按键 | 功能 |
|------|------|
| ← → | 切换轨道 |
| A D | 切换轨道 |
| ↑ W 空格 | 跳跃 |
| ↓ S | 快速下落 |

### 📱 手机
| 操作 | 功能 |
|------|------|
| 左右滑动 | 切换轨道 |
| 点击屏幕 | 跳跃 |

## 🎵 音乐选择

1. 点击灵动岛
2. 在"🎧"标签中选择
3. 点击曲目封面
4. 点击"✕"关闭
5. 点击播放按钮开始

## 👤 用户管理

### 查看信息
灵动岛 → 👤 用户标签

### 退出登录
灵动岛 → 👤 用户标签 → 🚪 退出登录

## ⚙️ 设置

### 调整音量
灵动岛 → 🔧 设置标签 → 拖动滑块

## 💡 快捷提示

- 🎯 未登录时会自动显示登录界面
- 🔄 刷新页面后自动恢复登录状态
- 🎵 游戏中点击灵动岛会暂停游戏
- ⌨️ 支持回车键快速提交表单
- 🎲 点击随机按钮快速选择曲目

## ⚠️ 注意事项

- 账号数据存储在本地浏览器
- 清除浏览器数据会删除账号
- 不同浏览器的账号独立
- 密码以明文存储，请勿使用重要密码
