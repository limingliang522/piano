# 🧪 画质系统测试指南

## 快速测试步骤

### 1. 启动游戏
打开 `index.html` 在浏览器中运行

### 2. 观察自动检测
- 等待 5 秒（300 帧）
- 查看控制台输出的检测结果
- 注意：浏览器类型、性能模式、目标 FPS

### 3. 查看性能统计
- 按 `P` 键打开性能面板
- 检查：FPS、像素比、阴影状态、材质类型

### 4. 测试手动切换
- 点击右下角齿轮按钮
- 尝试切换不同画质模式
- 观察画面变化和 FPS 变化

### 5. 测试自动升降级
- 选择"自动"模式
- 手动切换到高画质，等待 FPS 下降
- 观察是否自动降级
- 等待 FPS 恢复，观察是否自动升级

## 预期结果

- 自动模式：120Hz 高画质
- Ultra 模式：120-144Hz 超高画质

### 低端设备
