# 纯净原声输出配置

## 概述

音频系统已配置为**纯净原声输出模式**，最大程度保留钢琴采样的原始音色，只添加最轻微的空间感。

## 配置详情

### 1. 均衡器（EQ）
- **低频（250Hz）**: 0 dB（不增益）
- **中频（2000Hz）**: 0 dB（不增益）
- **高频（6000Hz）**: 0 dB（不增益）
- **目的**: 保持原始频率响应，不改变音色

### 2. 混响效果
- **混响时间**: 1.2 秒（小型房间）
- **干声比例**: 85%（原声为主）
- **湿声比例**: 15%（轻微混响）
- **目的**: 添加自然的空间感，避免过于干涩

### 3. 立体声增强
- **延迟时间**: 15ms（Haas 效果）
- **增强强度**: 20%（轻微）
- **目的**: 增加自然的立体声宽度

### 4. 主音量
- **增益**: 1.0（无额外增益）
- **目的**: 保持原始音量，不过度放大

### 5. 多段压缩器
- **低频**: 温和压缩（防止破音）
- **中频**: 适度压缩（保持清晰）
- **高频**: 轻微压缩（保持明亮）
- **Makeup Gain**: 1.0（不增益）
- **目的**: 只做保护性压缩，不改变动态

## 音频处理链路

```
音频源
  ↓
多段压缩器（保护性）
  ↓
均衡器（0 dB，透明）
  ↓
立体声增强（20%，轻微）
  ↓
混响分支
  ├─ 干声（85%）
  └─ 湿声（15%）
  ↓
主音量（1.0，无增益）
  ↓
输出
```

## 音质特点

✅ **保真度高**: 最大程度保留原始采样音色  
✅ **动态自然**: 不过度压缩，保持钢琴的自然动态  
✅ **空间适度**: 轻微混响增加空间感，不喧宾夺主  
✅ **立体声自然**: 轻微增强，不过度夸张  
✅ **音量合理**: 不过度放大，避免失真

## 调整建议

如果需要更多空间感，可以调整：
- 增加混响比例（15% → 25%）
- 延长混响时间（1.2s → 1.8s）

如果需要更干净的声音，可以调整：
- 减少混响比例（15% → 5%）
- 缩短混响时间（1.2s → 0.8s）

## 技术细节

### 混响脉冲响应
- 采样率：44100 Hz
- 缓冲长度：52920 采样（1.2 秒）
- 衰减模式：指数衰减（τ = 0.4）
- 早期反射：前 50ms
- 立体声宽度：15%

### 压缩器设置
- **低频**: Threshold -20dB, Ratio 4:1, Knee 30
- **中频**: Threshold -18dB, Ratio 3:1, Knee 30
- **高频**: Threshold -15dB, Ratio 2.5:1, Knee 25

## 性能影响

纯净原声模式相比 3D 空间模式：
- ✅ CPU 使用率更低（更短的混响）
- ✅ 内存占用更少（更小的脉冲响应）
- ✅ 延迟更低（更少的处理）
- ✅ 音质更纯净（更少的处理）
