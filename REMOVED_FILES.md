### audio-engine.js

- 删除了音色模拟系统（施坦威 EQ 链）
- 删除了多段压缩器

- 代码从 930 行减少到约 300 行

### game.js
- 更新了加载提示文本
- 更新了加载项数量（从 30 改为 12）
- 更新了相关注释

## 新的音频系统

现在使用 Web Audio API 的 OscillatorNode 生成正弦波：
- 无需加载外部文件
- 启动速度极快
- 内存占用极小
- 代码简洁清晰

如果需要恢复音色系统，可以：

2. 重新下载采样文件
3. 恢复 audio-engine.js 中的采样加载代码
